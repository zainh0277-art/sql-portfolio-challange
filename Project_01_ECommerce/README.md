# Retail Store Management Database

## Overview
A PostgreSQL database for managing retail store operations.

## Features
- User Management
- Employee Records
- Product Catalog
- Order Management
- Financial Tracking

## Database Design
- 6 Tables
- Primary Keys
- Foreign Keys
- Normalized up to 3NF
- Referential Integrity with ON DELETE CASCADE

## Technologies
- PostgreSQL
- SQL
- VS Code
